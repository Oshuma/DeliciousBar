DeliciousBar by Oshuma

OS X menu bar app to access your Delicious bookmarks.

Install

Download and uncompress to /Applications (or where ever) and run DeliciousBar.app. On first run, the preference window will pop up asking for your Delicious username and password. Save your credentials, then choose the 'Options → Sync' from the menu.

You can use the 'Sync on launch' option to automatically sync bookmarks when you start DeliciousBar.

Download

You can download the code for this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/Oshuma/DeliciousBar

get the source code on GitHub : Oshuma/DeliciousBar
